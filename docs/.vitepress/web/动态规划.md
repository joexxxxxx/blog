斐波那契数，通常用 F(n) 表示，形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。

```js
const cache = {}
var fib = function(n) {
  if (n === 0 || n === 1) return n;
  if (cache[n]) return cache[n]
  cache[n] = fib(n - 1) + fib(n - 2)
  return cache[n]
};
```
